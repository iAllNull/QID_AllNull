# QID_AllNull 1.0.0

> ✏ 一款简洁优雅的个人主页

## 开始使用

1. 下载主题：[发布版](https://github.com/iAllNull/QID_AllNull/releases)，注意是下载 VOID-x.x.x.zip 这个压缩包，而不是 Source code
2. 上次到服务器
3. 解压
